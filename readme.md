## Database Tables Used

- **employees**
- **projects**
- **employee_project**

These tables were populated and cleaned in previous tasks (Task 1 & Task 2).

---

## Task Objective
Write SQL queries to:

- Retrieve all rows from tables  
- Select specific columns  
- Filter data using conditions  
- Sort results  
- Use pattern matching with `LIKE`  
- Limit the number of rows  
- Perform `JOIN` operations  

---

##  Queries Included
The script contains:

1. Show all employee records  
2. Display employee names and salaries  
3. Employees belonging to the IT department  
4. Employees earning above 40,000  
5. Projects with valid start dates  
6. Salary-based ordering  
7. Limiting results  
8. Pattern matching (names starting with "D")  
9. Date range filtering  
10. Joining employees with their departments
